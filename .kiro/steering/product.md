# Product: Education Anime Generator

A full-stack AI platform that transforms educational topics into anime-style visual content, interactive browser simulations, 3D models, and storyified anime series. Designed as an API-first module that integrates into the existing CatchupXV1 LMS.

## Core Features

- **Anime Generation** — topic → anime-style scenes/characters/animations via Fal.ai (Animagine XL 4.0)
- **Simulation Engine** — topic → self-contained interactive HTML simulations via LLM code generation
- **3D Model Engine** — object name → GLTF 3D model via Fal.ai (Hunyuan3D-2.1)
- **Storyification** — topic → multi-episode anime series with narrative arc and sequential scenes
- **Bella** — persistent 3D VRM anime assistant with LLM chat, TTS voice, and lip sync

## Target Users

Students, educators, and e-learning platforms wanting visual/interactive learning experiences.

## Integration Target

Designed to be merged into CatchupXV1 as new routers/services/pages following its existing patterns.
